Práctica I: Análisis de Sistemas

Comparativa exhaustiva entre Regresión Paramétrica y Redes Neuronales Feedforward para el mercado inmobiliario.

Ingeniería de Sistemas Inteligencia Artificial Docente: Mg. Ing. Hugo Espetia

Enfoque Tradicional

Basado en el modelo de Regresión Lineal Múltiple. Este enfoque asume que el precio tiene una relación lineal con las variables independientes. Se define por la ecuación:

$$Y = \beta_0 + \beta_1X_1 + \beta_2X_2 + ... + \beta_nX_n + \epsilon$$

Donde \(\beta_n\) son coeficientes calculados mediante Mínimos Cuadrados Ordinarios.

Enfoque Inteligente

Utiliza una Red Neuronal Artificial (RNA) de tipo Perceptrón Multicapa. Su ventaja es la capacidad de modelar relaciones no lineales complejas mediante funciones de activación.

$$a^{(l)} = \sigma(W^{(l)}a^{(l-1)} + b^{(l)})$$

Implementado con la librería TensorFlow 2.15 y optimizador Adam.

Conclusiones del Estudio
